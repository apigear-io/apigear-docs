    temperature: float
    targetTemp: float

How It Works

Define your API interfaces in YAML with properties, methods, and signals. Choose a template for your target platform. Generate complete, buildable code with one command. Iterate on your API — regenerate anytime without losing your implementation.

Stateful APIs That Match Your Code

ObjectAPI models interfaces with observable properties, callable operations, and server-pushed signals. Write in a developer-friendly IDL or YAML — both generate the same code. Designed for systems where state matters: automotive, gaming, IoT.
